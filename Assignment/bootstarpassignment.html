Q.2 What is a Bootstrap Container, and how does it work?

Ans:-Containers are the most basic layout element in Bootstrap and are required when using grid system.

Container are used in many ways
  • container, which sets a max-width at each responsive breakpoint
    • container-fluid, which is width: 100% at all breakpoints
  • container-{breakpoint}, which is width: 100% until the specified breakpoint

Q.4 What do you know about the Bootstrap Grid System?

Ans:-

Bootstrap's grid system uses a series of containers, rows, and columns to layout and align content. It's built with flexbox and is fully responsive.

Q.5 What is the difference between Bootstrap 4 and Bootstrap 5

Ans:-

Bootstrap 4

  • Bootstrap 4 has a jQuary
  • It has 5 responsive tiers
  • Bootstrap 4 has limited color support.
  • Bootstrap 4 does not have its own SVG icons.

Bootstrap 5

  • It has removed jQuery and switched to JavaScript.
  • Bootstrap 5 has expanded color support.
  • In Bootstrap 5 you will find its own SVG icons.
  • It has 6 responsive tiers

Q.6 What is a Button Group, and what is the class for a basic Button Group?

Ans:- “Button Groups” in Bootstrap is a class of name “btn-group” which is used to create a series of buttons in groups (without spaces) vertically or horizontally.

for eg:-

<div class="btn-group">
<button type="button" class="btn">Click</button>
</div>

Q.8 In Bootstrap 4, what is flexbox?

Ans:-

In Bootstrap 4, flex box is used to control the layout and alignment specification of Bootstrap 4 components. This box makes it easier to design flexible responsive layout structures without using float or positioning attributes.
